# SmartphoneUse

This repository provides supplementary information for the article entitled, "Smartphone use for communication and learning among university students in Bangladesh" authored by Shahjahan et al. (2025+). This study explores the effects of smartphone use on communication and knowledge acquisition (i.e., learning).

The following files are provided under this repository:
1. Working_file_smartphone.sav -- An SPSS data file consisting of the entire survey data analyzed for this paper.
2. DataAnalysis_video.mp4 -- A MP4 video file that shows the data analysis.
3. R_code.R -- A file consisting of R code file that can be useful in reproducing the results presented.  

Note: Using the dataset and the video file, one should be able to reproduce all results presentend in the article. This article is currently under review.
